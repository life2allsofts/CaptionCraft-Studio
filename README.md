# CaptionCraft Studio 🎬

**Advanced Subtitle Generation Tool with AI-Powered Features**

*Bridging Geomatics Engineering and Software Engineering through AI*

## 🌟 Features

- 🎯 **Word-by-Word Timing**: Karaoke-style subtitle appearance
- 🎨 **Advanced Styling**: Professional CSS-based formatting
- 🤖 **AI Integration**: OpenAI Whisper for speech-to-text
- 📹 **Multi-Format Export**: VTT, SRT, SBV support
- 🖥️ **Modern GUI**: Dark-themed, user-friendly interface
- 🔧 **Cross-Platform**: Windows, macOS, Linux ready
